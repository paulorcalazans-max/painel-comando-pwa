# Painel de Comando — PWA

Aplicativo PWA local para estudo, revisão, musculação, corrida, sono e batimentos cardíacos.

## Como usar localmente

1. Extraia o ZIP.
2. Abra `index.html` no navegador.
3. Para instalar como PWA e usar cache offline, hospede em GitHub Pages, Cloudflare Pages, Netlify ou Vercel.

## Dados

Os dados ficam no próprio navegador, em `localStorage`.

Use a tela **Backup** para exportar/importar JSON.

## Observação sobre notificações

Notificações no navegador dependem de permissão e do comportamento do sistema operacional. O app também mostra lembretes internos na tela inicial.
